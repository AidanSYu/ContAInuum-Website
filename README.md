# ContAInuum Website

A modern, immersive website built with React, TypeScript, and Vite.

## Deployment

This project is configured for deployment on Vercel.

### Deploy to Vercel

1. Push your code to GitHub
2. Import your repository in Vercel
3. Vercel will automatically detect the configuration and deploy

The build configuration is already set up in `vercel.json`.

## Local Development

```bash
cd app
npm install
npm run dev
```

The site will be available at `http://localhost:5173`

## Build

```bash
cd app
npm run build
```

The production build will be output to `app/dist/`

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- GSAP (animations)
- Radix UI (components)
